

# Intel® Quartus® Prime Lite Edition (Version 18.1) Installation Guide

Follow these steps to download and install the Intel® Quartus® Prime Lite Edition Design Software Version 18.1 for Linux from the FPGA Software Download Center on Intel's website.

1. Visit the FPGA Software Download Center

Begin by navigating to the [FPGA Software Download Center at Intel.com](https://www.intel.com/content/www/us/en/collections/products/fpga/software/downloads.html?edition=lite&platform=windows&download_manager=direct). This page hosts various versions of FPGA design software, including the Quartus Prime Lite Edition.

2. Select the Software Version

Select Version 18.1 from the available software versions. In the dropdown menu, select version **18.1** 

![Intel® Quartus® Prime Lite Edition Design Software](/get-started/prime-lite.jpg)

> It's important to choose the correct version to ensure compatibility with your project requirements. 

3. Choose the Installation Package

Click on **Intel® Quartus® Prime Lite Edition Design Software Version 18.1 for Windows**. This option will redirect you to the download page specific to the Windows operating system version of the software. 

![Intel® Quartus® Prime Lite Edition Design Software](/get-started/software-type.jpeg)

4. Navigate to Multiple Download Options

Scroll down to the "Multiple Download" section. This area allows for downloading different components of the software in separate parts, if needed.

![Intel® Quartus® Prime Lite Edition Design Software](/get-started/multiple-download.jpeg)


5. Download the Software


Finally, download the zip file containing the installation package for Intel® Quartus® Prime Lite Edition Design Software Version 18.1. Ensure you have sufficient storage space and a stable internet connection before starting the download.


After downloading, proceed with the installation by unzipping the file and running the installer specific to Windows. Follow the on-screen instructions to complete the installation process.

6. Press Yes and click Next in the QuartusLite setup installation prompt.

![Intel® Quartus® Prime Lite Edition Design Software](/get-started/installation.jpg)

7. Accept the agreement and click Next again. Select the appropriate installation folder or retain the default directory. Make sure the following boxes are marked(modelsim and MAX 10 device support) for installation. And Press Next and let the installation complete 

![Intel® Quartus® Prime Lite Edition Design Software](/get-started/select-components.png)

---

Remember to check the system requirements and any additional instructions provided by Intel to ensure a smooth installation process.
