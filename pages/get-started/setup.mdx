## Setup a new project in Quartus Lite Software

1. Launch the Quartus  software, select File > New Project Wizard. The Introduction page opens

![Intel® Quartus® Prime Lite Edition Design Software](/get-started/new-project-wizard.png)

2. Enter the following information for working directory, name, and top-level entity for the project

![Intel® Quartus® Prime Lite Edition Design Software](/get-started/directory.png)

3. Select the Empty project template and click Next:  

4. Click Next twice and navigate to the Family, Device and Board Settings.  Select the specifics on the respective device target as mentioned below (Device – 10M50DAF484C7G). You can add the required Verilog files later or edit the files in the Quartus editor once the project has been setup.

![Intel® Quartus® Prime Lite Edition Design Software](/get-started/device.png)

5. Select the simulation Tool as ModelSim-Altera(note – this is not just ModelSim but the one with Altera) and format as Verilog HDL. Retain the other Tool selections to default. 

6. Review the new project summary page and make sure all the specifics that you entered are reflected. 

![Intel® Quartus® Prime Lite Edition Design Software](/get-started/summary.png)

7.  Quartus Lite Project navigator will now reflect the instance as your top level design entity under the device name.

![Intel® Quartus® Prime Lite Edition Design Software](/get-started/entity.png)
